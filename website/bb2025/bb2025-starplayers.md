# 2025 Star Players

Star Player "Special Rules" are listed outside their "Skill & Traits" section,
indicating that they are neither. But at the same time, the wording of many 
of them reads exactly as if they were skills or traits.

These differences, combined with (usually) short descriptions, open up for a lot
of interpretation.

This section describes the choices made by Jervis when required.

## Puggy Baconbreath - Hafling Luck
This special rule uses the exact same wording as the Pro skill, but without most
of the restrictions mentioned in Pro. Leaving it open to interpretation where it
actually works.

Some questions:

1. Does it work outside Puggy's Activation? Like when rolling for Landing.
2. Does it work on rolls, not on Puggy's behalf? Like Argue the Call.
3. Does it work on other player rolls?
4. Does it work if Puggy is not on the field?

All of these questions are unclear, but given that the description mirrors Pro, 
Jervis is using the interpretation that Hafling Luck behaves like a one-time
use of Pro.
