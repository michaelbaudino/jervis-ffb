# 2025 Rules

!!! warning "Under Active Development"
    This document should not be considered a full review of all rules, but 
    rather as a reference to the current state of how the rules are implemented. 
    Any decisions made in this document are subject to change.

In an ideal world, the Blood Bowl 2025 rulebook would be crystal clear with no 
ambiguities. Unfortunately, we do not live in that world.

This means that some things are not specified. Others are unclear or ambiguous.
In many cases, this forces Jervis to make a judgment call with regard to how 
to interpret them.

This document thus serves as a reference for the current state of the rules as 
implemented by Jervis.

For copyright reasons, we cannot reproduce large portions in the rulebook. 
Quotes will be used where appropriate, but otherwise we will just reference the 
relevant page in the rulebook.

The expectation is that Games Workshop will release FAQs or Erratas resolving 
many of these issues, but some will most likely never be fully answered.

To resolve rule disputes, Jervis is currently looking at three other
sources:

1. [NAF](https://www.thenaf.net/naf-recommendations-and-clarifications-for-bb2025/)
2. [FUMBBL](https://fumbbl.com/BB2025.pdf)
3. [Designer's Commentary and Errata](https://www.warhammer-community.com/en-gb/articles/uxzldbhu/blood-bowl-faq-update-play-ball-with-new-clarifications/) for the 2020 ruleset.

Based on these sources, Jervis will then make a decision. This decision plus
the rationale is documented in the subsections for this page. For each ruling,
the below boxes will be used to clarify the source of an argument.

!!! naf "NAF"

    This is the NAF interpretation of the 2025 rules.

!!! fumbbl "FUMBBL"

    This is the FUMBBL interpretation of the 2025 rules.

!!! bloodbowl "2020 Rules"

    This is how the 2020 ruleset (2nd edition) was designed. While strictly not
    relevant for the 2025 edition, if the 2025 version is ambiguous, we can use
    the 2020 version as a possible tie-breaker.

Currently, the primary goal for Jervis is to have _a implementation_ of the full
ruleset. However, given that different people will have different opinions on 
some of the choices made, the intent is to make contentious decisions 
configurable for a ruleset, allowing two players to play the game how they 
prefer.
